# タイマー&ストップウォッチアプリ

個人使用のためのタイマー&ストップウォッチアプリです。

## 機能

- ⏱️ ストップウォッチ
- ⏰ タイマー
- 🕐 現在時刻表示
- 📋 サーキットタイマー
- 🔔 多様なアラーム音
- ⏰ チクタク音
- 🎨 カスタマイズ可能
- 📱 PWA 対応

## 使用方法

1. ブラウザでアクセス
2. スマホの場合は「ホーム画面に追加」でアプリ化
3. 個人使用のみ

## 技術仕様

- HTML5 / CSS3 / JavaScript
- PWA (Progressive Web App)
- Service Worker
- Web Audio API
